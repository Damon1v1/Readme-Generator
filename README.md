# README Generator
## Application that generates a README from command line prompts

## Features
```
Accepted user inputs
Generated Markdown file
Table of contents
List of licenses to choose from
Title section
Description Section
Contact section
Contribution section
Credits section
License badge render
Intallation section
Testing Section
```
